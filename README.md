# Alpina_Digital_bot
Проект от компании «Alpina Digital»: API конструктор ботов на основе GPT

## Основные возможности
- **Управление ботами**: Полный CRUD (создание, чтение, обновление, удаление) для сущностей Боты, Сценарии и Шаги.
- **Гибкие сценарии**: Реализована система хранения состояний и промптов в формате **JSON** в базе данных.
- **Интеграция с GPT**: Поддержка библиотеки `openai` и логики ChatCompletion для генерации ответов.
- **Демо-режим**: Возможность тестирования API без реального ключа (имитация ответов GPT).
- **Админ-панель**: Удобный интерфейс Django для настройки контента.

## Технический стек
- **Backend**: Python 3.12, Django 6.0.3
- **API**: Django REST Framework (ViewSets, Serializers, Routers)
- **Database**: SQLite (ORM Django)
- **Интеграции**: OpenAI API

## Запуск
1. Клонируйте репозиторий: `git clone https://github.com/apollinaryaskiba-beep/Alpina_Digital_bot`
2. Установите зависимости: `pip install -r requirements.txt`
3. Примените миграции: `python manage.py migrate`
4. Загрузите демонстрационные данные (обязательно!): python manage.py loaddata initial_data
5. Запустите сервер: python manage.py runserver

## Тестирование API
После запуска сервера доступны следующие эндпоинты:

*   **Root API:** [http://127.0.0.1:8000]
*   **Список ботов:** `GET /api/bots/` — просмотр всех созданных ботов Alpina Digital.
*   **Шаги сценария:** `GET /api/scenarios/1/steps/` — просмотр всех этапов конкретного сценария.
*   **Запуск логики GPT:** `POST /api/steps/1/run/` — отправка сообщения GPT.

**Пример запроса для тестирования бота:**
В теле запроса отправьте JSON: {"input": "Ваш вопрос боту"}.
